команды Git'a по порядку

  • git clone (path to repository) - клонировать репозиторий в данную папку
  • git status - узнать статус нашего текущего состояния
  • git add - завести файл в систему гит(начать остлеживание)
  • git commit -am "" - закоммитить измнения(зафиксировать)

сообщения Git

  • no commits yet - пока что нет коммитов(не зафиксированы изменения)
  • Untracked files - не отслеживается файл Git'ом
  • change to be committed - измнения которые будут закоммичены(зафикисированы гитом)
  • changes not staged for commit -
